WebGPU Renderer

Geometra's WebGPU backend — solid boxes and text via a single GPU draw pipeline.
Checking WebGPU support…
What this demonstrates: The same declarative box() and text() tree you'd give the canvas renderer, rendered instead via a WebGPU pipeline. Text is rasterized into an offscreen canvas atlas and sampled as a GPU texture. Solid boxes are drawn as vertex-colored triangles.

Current gaps: border-radius, gradients, shadows, images, selection highlights, and focus rings. Text rendering works for any PDF-base-14 or web font present in the browser.
